## Start frontend

From `frontend/`:

```bash
npm install
```

Create/update `frontend/.env`:

```env
VITE_API_BASE_URL=http://localhost:8000
```

Run frontend:

```bash
npm run dev
```

App URL: [http://localhost:5173](http://localhost:5173)

## Useful commands

From `frontend/`:

```bash
npm run test
npm run test:e2e
```
